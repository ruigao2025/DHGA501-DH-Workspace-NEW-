# Rethink

## 以张隐韬日记为核心史料，运用社会网络分析方法，探究近代精英人际网络，论证黄埔军校早期是否真正突破传统地缘政治结构（"地方主义"）

### 核心价值
- **一手史料**：张隐韬日记提供直接的个体（微观）视角  
- **方法论创新**：SNA实现关系结构量化分析  
- **历史意义重大**：直击近代化进程进程关键问题

### 当前局限
- **史料处理低效**：手工转录、校对、编码工作冗杂  
- **网络数据主观**：人工提取易受研究者影响  
- **背景知识受限**：本人学识限制了解读深度和广度  

### **AI能力**<br>
- OCR、命名实体识别、关系抽取<br>
  手写日记数字化转录<br>
  自动标记人物、地点、组织实体<br>
  智能 智能识别互动关系类型及权重<br>
- 知识图谱、大语言模型、信息检索<br>
  构建黄埔军校专属知识图谱<br>
  LLM辅助问答与假说生成<br>
  多源史料自动交叉验证<br>

### 技术架构
- 史料输入层<br>
    ├─ 张隐韬日记（扫描件）<br>
    └─ 补充史料（军校名录、档案）<br>
    
- 数据预处理层<br>
    ├─ OCR服务（文字识别）<br>
    └─ NLP流水线（实体关系提取）<br>
    
- 数据存储层<br>
    ├─ 图数据库 <br>
    └─ 关系数据库 <br>
    
- AI分析层<br>
    ├─ 网络建模引擎<br>
    ├─ 社区检测算法<br>
    └─ 知识图谱融合<br>
    
- 模拟可视化层<br>
    └─ 网络可视化<br>
    
- 用户界面层

### 研究方法流程

- 数据采集：多源史料数字化集成
- AI处理：自动提取实体关系构建网络
- 高级分析：GNN+聚类+模拟综合研判
- 成果输出：交互可视化 + 量化指标 + 综合分析报告
